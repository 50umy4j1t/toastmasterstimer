# Toastmasters Timer

## Overview  
This project includes implementations for a Timer used during Toastmasters events. It contains versions in both Python and HTML/JavaScript.

## Python Version  
You can run the Python version of the timer using the following command:
```bash
python timer.py
```
### Deployment  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/50umy4j1t/toastmasterstimer.git
   ```  
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application.

## HTML/JavaScript Version  
The HTML/JavaScript version can be run directly in your web browser. Just open `index.html` to use the timer.
### Deployment  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/50umy4j1t/toastmasterstimer.git
   ```  
2. Open `index.html` in a web browser.

## Usage Instructions  
The timer starts counting up from 0. You can use the buttons to control the timer:
- Start: Begins the timer.
- Stop: Halts the timer.
- Reset: Clears the timer back to 0.
